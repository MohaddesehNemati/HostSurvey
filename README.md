# Streamlit Dashboard | Host Survey (Calendar Closure)

## Fix cache issues
If you upgraded the code and see KeyError / missing columns:
- In the app sidebar click: **پاک‌کردن کش (Fix errors)**

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy (Streamlit Community Cloud)
- Upload this repo to GitHub
- In Streamlit Community Cloud: New app → select repo → main file `app.py` → Deploy

## Data
Main dashboard data:
