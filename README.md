# Dashboard | بستن تقویم

این ریپو یک داشبورد Streamlit برای تحلیل «بستن تقویم در تعطیلات ۲۱ تا ۲۴ بهمن» است.

## اجرا روی سیستم خودت
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy آنلاین (Streamlit Community Cloud)
1) این پوشه را در GitHub آپلود کن.
2) در Streamlit Community Cloud گزینه New app را بزن.
3) Repo و branch را انتخاب کن و فایل اصلی را `app.py` قرار بده.
4) Deploy.

## داده‌ها
فایل اصلی برای داشبورد:
- `data/تحلیل_بستن_تقویم_هفته_گذشته.xlsx`

در صورت نیاز، دو فایل خام هم داخل `data/` هستند.
